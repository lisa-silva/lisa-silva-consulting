# Lisa Silva Consulting

A complete, responsive consulting website built with Next.js App Router, TypeScript, and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm start
```

The contact form currently demonstrates a complete front-end success flow. Connect the submit handler in `components/contact-form.tsx` to an email or form service before production launch.
